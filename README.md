Voting Management System

A modern web-based Voting Management System built with Vue 3, Vite, Pinia, and Tailwind CSS.

This project provides an administrative dashboard for managing voters, candidates, and monitoring election results through a clean and responsive interface.

🚀 Tech Stack

Frontend: Vue 3 (Composition API)

Build Tool: Vite

State Management: Pinia

Routing: Vue Router

Styling: Tailwind CSS

Package Manager: npm
